        我们公司是一家与国内重点院校专家教授和科研院所专家教授长期合作，以中国材料科工集团苏州非矿院防水设计研究所专家教授为技术依托，集防水材料科研、开发、生产销售、工程承包、技术咨询为一体的专业防水公司，是国内早期研制和生产聚合物水泥防水材料的生产企业。公司技术力量雄厚、生产管理规范、配方独特、工艺先进、常规检测设备齐全。
